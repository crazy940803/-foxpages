DATABASE TABLES - FPS


Table: SERVERS (Fields: 20)

This table contain the servers configurations.

Name Type Length Decimals Nulls Default Comments
ID C 4 0 No - Server ID. (e.g. 0001,0002)
NAME C 10 0 No - Server name. (e.g. Server001, Server002)
DESCRIPTION C 50 0 No - Server description. (e.g. HTTP Server, HTTPS Server)
IP C 48 0 No - Server listenner IP address. (e.g. 0.0.0.0 to all IPs, 200.130.10.2 to bind to a specific IP)
PORT N 5 0 No - Server listenner port number. (e.g. 80 to HTTP, 443 to HTTPS)
TYPE C 6 0 No - Server type:
HTTP - Hipertext Transfer Protocol
FCGI - FastCGI Protocol
TUNNEL - Tunnel Entry Points
TUNNEX - Tunnel Exit Points
BANDWIDTH N 6 0 No - HTTP bandwidth (kilobytes).
CHUNKED N 1 0 No - Enables HTTP chunked transfer encoding:
0 - Disabled
1 - Enabled
COMPRESSION N 1 0 No - Enables compression. (Used for TUNNEL and TUNNEX server types only)
0 - Disabled
1 - Enabled
KEEPALIVE N 1 0 No - Enables HTTP connections keep alive:
0 - Disabled
1 - Enabled.
PASSWORD C 32 0 No - Enables AES 256 encryption. Must be 32 bytes long. (Used for TUNNEL and TUNNEX server types only)
LOGLEVEL N 1 0 No - Enables debug logging:
0 - Disabled
1 - Server events
2 - Server and threads events
3 - Server, threads and processing events.
LOGREQUESTS N 1 0 No - Enables server requests logging:
0 - Disabled
1 - Enabled
REMOTEHOST C 50 0 No - Remote host. (Used for TUNNEL and TUNNEX server types only)
REMOTEPORT N 5 0 No - Remote port. (Used for TUNNEL and TUNNEX server types only)
SECURE N 1 0 No - Enables secure connections (HTTPS):
0 - Disabled
1 - Enabled
CERTIFICATENAME C 50 0 No - Certificate name or common name (CN). (e.g. www.example.com)
CERTIFICATESTORE C 50 0 No - Certificate store or file name.
CERTIFICATEPASSWORD C 50 0 No - Certificate file password. Used for certificate files stored in certificates folder.
STATE N 1 0 No - Server state:
0 - Disabled
1 - Enabled
Indexes:
Label Primary Key
ID Yes ID


Table: SITES (Fields: 8)

This table contain the web sites configurations.

Name Type Length Decimals Nulls Default Comments
ID C 4 0 No - Site ID. (e.g. 0001,0002)
SERVER C 4 0 No - Server ID from SERVERS table.
NAME C 50 0 No - Site name. (e.g. Site001, Site002)
DESCRIPTION C 50 0 No - Site description. (e.g. PHPBB Forum, RoundCube Webmail)
HOSTNAME C 100 0 No - The Host header received from HTTP request. (e.g. forum.example.com, webmail.example.com)
INDEX C 20 0 No - Site index page. (e.g. index.fxp, index.php, index.html)
DIRECTORY C 100 0 No - The full path to the folder that contains the site files. (e.g. c:\sites\phpbb, c:\sites\roundcube)
REDIRECT C 100 0 No - The location to redirect (e.g. http://example.com, https://example.com)
Indexes:
Label Primary Key
ID Yes ID
SERVER No SERVER
HOSTNAME No HOSTNAME


Table: GATEWAYS (Fields: 10)

This table contain the gateways configurations.

Name Type Length Decimals Nulls Default Comments
ID C 4 0 No - ID. (e.g. 0001, 0002)
SERVER C 4 0 No - Server ID from SERVERS table.
NAME C 50 0 No - Name. (e.g. Gateway001, Gateway002)
DESCRIPTION C 50 0 No - Description. (e.g. FastCGI PHPBB Forum, FastCGI RoundCube Webmail)
HOSTNAME C 100 0 No - The Host header received from HTTP request. (e.g. forum.example.com, webmail.example.com)
URI C 100 0 No - The server will search the content of this field in the requested URI to determine when to send it to the gateway. (e.g. fill with ".php" to send every URI requests containing .php to the gateway)
PROTOCOL C 6 0 No - Gateway protocol:
FCGI - FastCGI
HOST C 50 0 No - Remote host address or IP.
PORT N 5 0 No - Remote port.
DIRECTORY C 100 0 No - The full path to the folder that contains the site files. Should be the same of SITES table. (e.g. c:\sites\phpbb, c:\sites\roundcube)
Indexes:
Label Primary Key
ID Yes ID
SERVER No SERVER
URI No URI
HOSTNAME No HOSTNAME
